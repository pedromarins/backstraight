Dúvidas sobre a parceria

Como funciona o modelo de parceria? +
Você revende o BackStraight como parte da sua oferta de consultoria. Cobramos por colaborador monitorado com assinatura mensal. Você define seu preço final e sua margem. Sem custo de setup, sem investimento inicial.
Preciso de equipe técnica para implementar? +
Não. A instalação é simples — um aplicativo desktop. A calibração leva 2 minutos por colaborador. Não exige treinamento técnico, hardware especial ou infraestrutura de TI.
Os relatórios vêm com minha marca? +
Estamos desenvolvendo opções de white-label para consultores parceiros. Na versão atual, os PDFs são gerados com a marca BackStraight. Entre em contato para discutir customização.
Como apresento isso para o RH do meu cliente? +
Use o argumento da NR-1 (vigência maio/2026) e o custo de não-conformidade (multas até R$6.708). Mostre que o BackStraight é science-based, 100% LGPD compliant (dados nunca saem do dispositivo) e gera relatórios automáticos para AEP/AET. Oferecemos materiais de apoio para consultores parceiros.
O sistema é seguro para ambientes corporativos? +
100%. Toda a IA roda localmente no dispositivo do colaborador. Nenhuma imagem é enviada, armazenada ou compartilhada. A câmera desliga ao pausar. Não requer conexão com internet para monitorar. Compliance total com a LGPD.
Qual o volume mínimo para ser parceiro? +
Não exigimos volume mínimo para começar. Você pode iniciar com um piloto em um cliente e escalar à medida que os resultados aparecem. Os dados de evolução do piloto são o melhor argumento de venda para os próximos clientes.
Funciona no Windows? A maioria dos meus clientes usa Windows. +
Sim. O BackStraight funciona em macOS e Windows. A maioria dos ambientes corporativos brasileiros usa Windows e o aplicativo é totalmente compatível. Necessita apenas de uma webcam.
